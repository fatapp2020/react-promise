# Desafío: Catálogo con MAPS y Promises


### Clonar Proyecto Primera Vez
```
git clone https://fatapp2020@github.com/fatapp2020/react-promise.git
```

### Subir Proyecto
```
- git init
- git add .
- git commit -m ""
- git push -u origin main
```

### Actualizar Proyecto Último Commit
```
- git pull
```

### Levantar Proyecto
```
- npm start
```
